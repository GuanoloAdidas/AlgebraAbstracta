Párrafo

Los sistemas que usan dos claves separadas, una para encriptar y otra para decriptar, se conocen como criptosistemas de clave pública (public key cryptosystems). Como el conocimiento de la clave de encriptación no le permite a nadie adivinar la clave de decriptación, la primera se puede hacer pública. Un criptosistema de clave pública le permite tanto a \(A\) como a \(B\) enviar mensajes a \(C\) usando la misma clave de encriptación. Culquiera puede encriptar mensajes para enviarselos a \(C\text{,}\) pero solo \(C\) sabe como decriptar estos mensajes.
